
# Laravel-react

#### 1 - Steps

Run :   

1 - ```npm install```     
2 - ```composer install```   
3 - ```php artisan key:generate```    
4 - ```php artisan jwt:secret```    
5 - ```npm run dev```   
6 - ```php artisan serve```     

##### ressources -> assets -> js -> components -> Main.js   
Define your routing for the react-router.

##### ressources -> assets -> js -> components 
All your react components.
